JavaScript Basics to Advanced

Learn JavaScript from basics to advanced concepts with examples!

1. JavaScript Basics

JavaScript is a programming language that allows you to add interactivity to websites. It is the third core technology of the web along with HTML and CSS.

Variables

In JavaScript, variables can be declared using `var`, `let`, or `const`:


let name = "John";
const age = 30;
var city = "New York";

Data Types

  • String: Text, e.g., "Hello"
  • Number: Integer or floating point numbers, e.g., 10, 5.5
  • Boolean: True or false values
  • Object: A collection of properties, e.g., `{ name: "John", age: 30 }`
  • Array: A list of values, e.g., `[1, 2, 3, 4]`
  • Null: Represents the intentional absence of a value
  • Undefined: A variable declared but not assigned a value

2. Functions

A function is a block of code designed to perform a particular task. Functions are reusable.

Function Declaration


function greet(name) {
    return "Hello, " + name + "!";
}

Function Expression


const greet = function(name) {
    return "Hello, " + name + "!";
};

3. Control Flow

Control flow in JavaScript determines the order in which code is executed.

Conditionals


if (age > 18) {
    console.log("Adult");
} else {
    console.log("Minor");
}

Loops

  • for loop: Repeats a block of code a specified number of times.
  • while loop: Repeats a block of code while a condition is true.
  • for...of loop: Loops over iterable objects (arrays, strings, etc.).

Example:


for (let i = 0; i < 5; i++) {
    console.log(i);
}

4. Objects and Arrays

Objects and Arrays are the two main data structures in JavaScript. Objects store key-value pairs, while Arrays store ordered lists.

Objects


const person = {
    name: "John",
    age: 30,
    greet: function() {
        return "Hello " + this.name;
    }
};

Arrays


const numbers = [1, 2, 3, 4, 5];
numbers.push(6);
console.log(numbers);

5. DOM Manipulation

DOM (Document Object Model) manipulation allows you to interact with the structure of the HTML document.

Getting Elements


let element = document.getElementById("myElement");
let elements = document.getElementsByClassName("myClass");

Changing Elements


element.innerHTML = "New content!";

Event Listeners


button.addEventListener("click", function() {
    alert("Button clicked!");
});

6. Advanced JavaScript Concepts

These topics delve into deeper aspects of JavaScript, including closures, promises, and asynchronous programming.

Closures

A closure is a function that remembers its lexical environment, even when the function is executed outside that environment.


function outer() {
    let counter = 0;
    return function inner() {
        counter++;
        console.log(counter);
    };
}

const increment = outer();
increment();  // Output: 1
increment();  // Output: 2

Promises

Promises are used to handle asynchronous operations in JavaScript.


let promise = new Promise((resolve, reject) => {
    let success = true;
    if (success) {
        resolve("Success!");
    } else {
        reject("Failure!");
    }
});

promise.then(result => {
    console.log(result);
}).catch(error => {
    console.log(error);
});

Async/Await

Async/await simplifies working with promises by allowing you to write asynchronous code in a synchronous manner.


async function fetchData() {
    let response = await fetch('https://api.example.com/data');
    let data = await response.json();
    console.log(data);
}

fetchData();
