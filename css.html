CSS Basics to Advanced

Learn CSS from scratch to advanced techniques with examples!

1. CSS Basics

CSS (Cascading Style Sheets) is used to style HTML elements. It allows you to control the layout, color, fonts, spacing, and more.

CSS Syntax

The syntax consists of selectors, properties, and values:


selector {
    property: value;
}

Example:


h1 {
    color: #333;
    text-align: center;
}

Selectors

  • Element Selector: Selects all elements of a specific type (e.g., `p`, `h1`).
  • Class Selector: Selects elements with a specific class (e.g., `.class-name`).
  • ID Selector: Selects an element with a specific ID (e.g., `#id-name`).

2. Box Model

The CSS box model defines the rectangular boxes generated for elements, consisting of the content, padding, border, and margin.

Components of the Box Model:

  • Content: The actual content of the box (text, images, etc.).
  • Padding: Space between the content and the border.
  • Border: Surrounds the padding (if any) and content.
  • Margin: Space outside the border, creating distance between elements.

Example:


div {
    width: 200px;
    padding: 20px;
    border: 5px solid #000;
    margin: 10px;
}

3. Positioning

CSS positioning allows you to position elements on the page in various ways. Here are the key positioning types:

  • Static: Default value, elements are positioned based on normal document flow.
  • Relative: Positioned relative to its normal position.
  • Absolute: Positioned relative to the nearest positioned ancestor.
  • Fixed: Positioned relative to the browser window.
  • Sticky: Behaves like relative until the scroll reaches a defined point, then behaves like fixed.

Example:


div {
    position: relative;
    top: 10px;
    left: 20px;
}

4. Flexbox

Flexbox is a layout model that allows items to be aligned and distributed within a container, even when their size is unknown or dynamic.

Flex Container and Flex Items

  • display: flex: Defines a flex container.
  • justify-content: Aligns flex items horizontally (e.g., `center`, `space-between`).
  • align-items: Aligns flex items vertically (e.g., `center`, `flex-start`).

Example:


.container {
    display: flex;
    justify-content: space-between;
    align-items: center;
}
.item {
    flex: 1;
}

5. CSS Grid

CSS Grid Layout is a powerful tool for creating 2D layouts. It allows you to define both rows and columns.

Grid Container and Grid Items

  • display: grid: Defines a grid container.
  • grid-template-columns: Defines the columns of the grid.
  • grid-template-rows: Defines the rows of the grid.

Example:


.container {
    display: grid;
    grid-template-columns: 1fr 2fr;
    grid-template-rows: 100px auto;
}
.item {
    grid-column: 1 / 2;
    grid-row: 1 / 3;
}

6. Responsive Design

Responsive design ensures that a website works well on all screen sizes, from mobile to desktop.

Media Queries

  • @media: A CSS rule that applies styles based on the device's screen size.

Example:


@media (max-width: 768px) {
    .container {
        grid-template-columns: 1fr;
    }
}

7. Advanced CSS

This section covers advanced topics like transitions, animations, CSS variables, and pseudo-classes.

Transitions

CSS transitions allow you to change property values smoothly (over a given duration).

Example:


button {
    background-color: #4CAF50;
    transition: background-color 0.3s ease;
}

button:hover {
    background-color: #45a049;
}

Animations

CSS animations allow you to create dynamic changes to elements.

Example:


@keyframes example {
    from {background-color: red;}
    to {background-color: yellow;}
}

div {
    animation: example 4s infinite;
}

CSS Variables

CSS Variables allow you to store values and reuse them throughout the stylesheet.

Example:


:root {
    --main-color: #4CAF50;
}

button {
    background-color: var(--main-color);
}
