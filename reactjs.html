Introduction

React.js is a JavaScript library for building user interfaces. It allows you to create reusable components and manage the state of your application efficiently.

Official Documentation

Prerequisites

  • Familiarity with ES6+ features (arrow functions, destructuring, etc.)

Getting Started

To start working with React, you need to set up a development environment. Use the following steps:

  1. Install Node.js: Download Node.js
  2. Install a code editor like Visual Studio Code
  3. Initialize a new React app:
    4. npx create-react-app my-app
  4. Navigate to the project folder and start the app:
    5. cd my-app
    npm start

3. Props

Props (short for properties) allow you to pass data to components.

function Greeting(props) {
  return <h1>Hello, {props.name}!</h1>;
}

Learn more: Props Documentation

4. State

State is used to manage the internal data of a component.

import React, { useState } from 'react';
function Counter() {
  const [count, setCount] = useState(0);
  return (
    <div>
      <p>Count: {count}</p>
      <button onClick={() => setCount(count + 1)}>Increment</button>
    </div>
  );
}
