Introduction to HTML

HTML (Hypertext Markup Language) is the standard language for creating web pages. It is used to structure content on the web, such as text, images, links, and multimedia.

Basic HTML Tags

HTML uses tags to create elements on the page. Here are the most commonly used basic tags:

  • <!DOCTYPE html> - Defines the document type and version of HTML.
  • <html> - The root element that contains the entire HTML document.
  • <head> - Contains metadata such as the document title, character set, and linked stylesheets.
  • <body> - Contains the main content of the web page.
  • <title> - Specifies the title of the webpage, shown in the browser tab.
  • <h1> to <h6> - Heading tags used for defining different levels of headings.
  • <p> - Paragraph tag, used to group text into separate blocks.
  • <a> - Anchor tag, used to create hyperlinks.
  • <img> - Used to display images on the webpage.
  • <ul>, <ol> - Lists: unordered (bullets) and ordered (numbered).
  • <li> - List items inside an unordered or ordered list.

Example:

<html>
    <head>
        <title>My First Webpage</title>
    </head>
    <body>
        <h1>Welcome to My Webpage</h1>
        <p>This is a simple paragraph of text</p>
        <a href="https://example.com">Click here to visit a website</a>
        <ul>
            <li>Item 1</li>
            <li>Item 2</li>
        </ul>
    </body>
</html>

HTML Attributes

Attributes provide additional information about HTML elements. Here are some common attributes:

  • href - Specifies the URL for an anchor tag (<a>).
  • src - Specifies the path for an image tag (<img>).
  • alt - Provides alternative text for images.
  • class - Specifies a class for CSS styling.
  • id - Specifies a unique identifier for an element.
  • style - Inline CSS styling for an element.
  • target - Specifies where to open the linked document (e.g., in a new tab).

Example:

<a href="https://example.com" target="_blank">Visit Example</a>
<img src="image.jpg" alt="Description of image">

HTML Forms

Forms allow users to input data. Here's an example of a simple HTML form:

  • <form> - The container for all form elements.
  • <input> - Allows the user to enter data. Type attribute defines input type (text, password, etc.).
  • <button> - Used to submit the form.

Example:

<form action="/submit" method="POST">
    <label for="name">Name:</label>
    <input type="text" id="name" name="name">

    <label for="email">Email:</label>
    <input type="email" id="email" name="email">

    <button type="submit">Submit</button>
</form>

Semantic HTML Elements

Semantic elements help with accessibility and SEO by clearly defining the structure of the page. Common semantic elements include:

  • <header> - Defines the header section of a page or a section.
  • <footer> - Defines the footer section of a page or a section.
  • <article> - Represents a self-contained piece of content.
  • <section> - Represents a section of related content.
  • <nav> - Defines a navigation menu or links.

Example:

<header>
    <h1>Website Header</h1>
</header>

<section>
    <h2>Section Title</h2>
    <p>This is a section</p>
</section>

<footer>
    <p>Website Footer</p>
</footer>

Advanced HTML Tags

Here are some advanced HTML tags and concepts that you'll need as you progress:

  • <iframe> - Embeds another document within the current page.
  • <canvas> - Allows you to draw graphics using JavaScript.
  • <video> and <audio> - Embeds media files.
  • <meta> - Defines metadata for the document, such as charset or keywords.

Example:

<iframe src="https://www.example.com" width="600" height="400"></iframe>

<video controls>
    <source src="movie.mp4" type="video/mp4">
    <source src="movie.ogg" type="video/ogg">
    Your browser does not support the video tag.
</video>
